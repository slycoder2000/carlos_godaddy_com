Arithmetic

The basic mathematical functions work accordingly: '+', '-', '*', '/'.

the modulo operator is used to determine the remainder '%'

for example: console.log(10%2); // -> produces 0

console.log(10%3); // -> produces 1

Strings

Newlines are used in strings with the backslash+'n' as such: "\\n".

backticks (template literals) are used whenever possible to use variables and perform multiple newlines '`'

Example: `half of 100 is ${100/2}` // -> result: half of 100 is 50

Unary Operators

some words are used to perform operations

Example of typeof: console.log(typeof 4.5); // produces -> number

Example of typeof: console.log(typeof "x"); // produces -> string

Binary Operators

the minus sign is an example of this - as make the following negative or subtract x from y.

Example: console.log(-(10-2)); // -> -8

Boolean Values

The basic boolean values are used: '<', '>', '!=', '==' (note double equal is used) or precisely equal '==='

Also NaN is used, and can be detected with the operator isNaN('1')

Logical Operators

The basic logical operators are '&&' (and) and '||' (or).

Ternary operator

Example: console.log(true ? 1 : 2);

console.log(false ? 1 : 2);

Empty Values

null and undefined are used to determine empty values

Automatic Type Conversion

to avoid type coercion, use precisely equal '==='
